mentogdut

    a console application for linux-based OS and all versions of windows, is a light-weight drcom-client-replacement that enables you to share your network without any limitations.

Download:
    mentogdut_linux_windows_x64_20161217.zip
Usage:
    mentogdut -s svrip [-p port] [-f pppoe-flag] [-k keepalive2_flag] [-m checksum]
    mentogdut -c conf
Options:

-s,--server <ip>the auth server ip
[-p,--port <port>]the auth server port [default: 61440]
[-f,--pppoe_flag <flag>]the pppoe flag [default: 6a]
[-k,--keep_alive2_flag <flag>]the keep alive2 flag [default: dc]
[-m,--checksum <mode>]the checksum mode [default: 0 automatic]
[-c,--config <file>]the path to configuration file
[-h,--help]give this help

mentogdut for router

    Software package for OpenWRT-based router with LuCI management interface included, can be installed either by the opkg utility or the LuCI.
    *Those packages has been tested with phicomm K2, MT7620, pandorabox 16.10, but it still not guaranteed to be compatible with yours.
    *Upgrade firmware at your own risk. Wrong firmware upgrading may damage your device and avoid the warranty.
